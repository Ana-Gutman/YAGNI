# ADR005: Redis para OTPs Temporales
**Estado:** Aceptada  
**Participantes:** Federica Cabrera y Ana Gutman  

## Contexto
Las claves OTP deben ser únicas, seguras y expirar rápidamente para garantizar la seguridad en el acceso a los refrigeradores inteligentes.

## Decisión
Utilizar Redis para la generación, almacenamiento y validación de OTPs con un TTL de 30 segundos.

## Opciones Consideradas
- **Redis:** Aceptada por su rapidez y soporte nativo para TTL.
- **Base de datos relacional:** Rechazada debido a mayor latencia y sobrecarga innecesaria.

## Consecuencias
- **Beneficios:** Garantiza rapidez, simplicidad y seguridad para la gestión de los OTPs de acceso a los refrigeradores.
- **Costos:** Introduce una dependencia adicional en Redis para esta funcionalidad.
